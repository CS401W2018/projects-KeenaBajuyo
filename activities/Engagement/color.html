Engagement - CSS Selectors

List out the different ways to select an item.

  1. Tag Name (Type Selector)
  2. Descendent Selector
  3. Child Selector
  4. ID Selector
  5. Class Selector

What is the difference between an ID and a Class?

An ID is used to identify an element. It's an attribute that can be used only once and should be unique, not duplicated. Generally, we should use a noun to name it.

A Class can be assigned to more than one tag. One tag can have more than one class. Generally, we should use an adjective to name it.

What is the difference between a SPAN tag and a DIV tag?

A SPAN tag doesn't show any changes on the webpage by itself and doesn't cause a line break. You can assign a class to it and use it within a sentence. In contrast, a DIV tag causes a line break and is used to group larger sections of content.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

A DIV tag wraps the content without offering any semantic value.

A HEADER tag defines introductory content or navigational links.

A MAIN tag carries the main content of the document.

A FOOTER tag declares the footer area, usually with metadata or links.

What are the different ways to select a color?

  • Name
  • RGB (You need to choose a color by specifying red, green, and blue values from 0 to 255)
  • RGBA (RGB with an alpha value for transparency)
  • Hex code (Two numbers for each color: red, green, and blue)
  • Short Code
  • HSL (Hue, Saturation, Lightness)
