Questions:

What is normal flow in HTML?

A normal flow signifies elements that is displayed so that they become visible in the code, with block elements piled up vertically and inline elements flowing horizontally.

What does float do? Why do you need to clear the float?

Float lets an element be positioned either left or right of its container, with other content circling around it. Clearing it ensures that subsequent elements start below the floated element, stopping them from wrapping around it.

What do absolute, fixed, and relative do?

Absolute positions an element relative to its nearest positioned ancestor (or the initial containing block if none exists), removing it from the normal flow.

Fixed positions an element relative to the viewport, meaning it will STAY in the same place even when the page is scrolled.

Relative positions an element relative to its normal position, without removing it from the normal flow.

How can you make nice navigation buttons for mobile devices?

To produce navigation buttons that is mobile-friendly it needs to be responsive, touch-friendly, simple, and visually appealing with clear labels and appropriate spacing.
