Explain the difference between an absolute link and a relative link.

An absolute link contains the full URL. Typically use to direct user to the different webpage.

A relative link provides a path relative to the current page's location. It links resources from the same website making it easier to locate.

Why would you use relative links?

It is a simplier way to direct users, provides shorter URL's, it is also very convenient for when you wanna improve some spots in your website because it is more localized in terms of the file and also it builds consistency.

List the 3 rules with relative links.

  1. Link using the same folder.
  2. Use ../ to move up one directory level. You can chain multiple ../ to move up multiple levels.
  3. Identify the path from the present directory to the aimed file.
