Explain the difference between an absolute link and a relative link.

An absolute link contains the full URL. It is typically used to direct the user to a different webpage.

A relative link provides a path relative to the current page's location. It links resources from the same website, making them easier to locate.

Why would you use relative links?

It is a simpler way to direct users, provides shorter URLs, and is also very convenient for when you want to improve some spots on your website because it is more localized in terms of the file. Additionally, it builds consistency.
