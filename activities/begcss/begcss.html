Explain the difference between inline, embedded, and external CSS.

Inline CSS is written directly in the HTML element's style attribute.

Embedded CSS is placed within a style tag in the HTML document's head.

External CSS is written in a separate .css file and linked to the HTML document.

Explain why you should only use external CSS.

By using external CSS, it will help my HTML files become cleaner and more organized. It also makes it easier to update styles across multiple pages and improves loading times by allowing browsers to cache the CSS file.

What is a selector used for?

A selector targets specific HTML elements to apply styles to them.
