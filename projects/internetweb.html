Internet

The internet is a broad network that connects everyone using computers and phones from around the world.

Web

Most commonly known as the 'World Wide Web,' it is a compilation of different webpages that are linked together through a web browser using the internet.

How are the Internet and Web different?

The internet is the network that connects all devices around the world, while the web is a service that runs because of the internet. It's like the internet is the root, while the web is its branches.

ASCII and Unicode

ASCII stands for American Standard Code for Information Interchange. It is the basis for encoding text in computers and other electronic devices, assigning a unique number to each character. Unicode is a system for encoding international characters, giving each one a special number. It's used to represent many languages and scripts, especially older ones like Latin, Arabic, Cyrillic, and Greek.

What is UTF-8?

UTF-8 is a particular way of encoding a Unicode code using a variable number of bytes, making it efficient to store different types of characters.

URL

URL stands for Uniform Resource Locator. It's basically the address of a webpage or a file on the internet. It is a unique way to let your browser locate and access any resource. It's like a physical address for a website on the internet.

Domain Name

A domain name is a simple and easy way to remember the address of a webpage as we type it into the web browser and access it.

HTTP

HTTP stands for Hypertext Transfer Protocol. It is a transmission protocol that lets us enter and connect with different web resources. It serves as the foundation of the web to load web pages.

HTTPS

HTTPS stands for Hypertext Transfer Protocol Secure. It's the secure version of HTTP. What it does is use encryption to guard data passed between a website and a web browser.

IP Address

IP Address stands for Internet Protocol (IP) address. It is a special number that recognizes every device connected to the internet.

DNS

DNS stands for Domain Name System. It is the most important part of the internet as it converts user-readable domain names into a machine-readable format that browsers use to load pages.

W3C Recommendations

A W3C recommendation refers to a standard or regulation advocated by the World Wide Web Consortium. It lays down the technical specifications and best practices for web technologies to enhance consistency and compatibility on the internet.

XML, XHTML, HTML, HTML5

XML stands for eXtensible Markup Language. Similar to HTML, it is designed for storing and transporting data, is self-descriptive, and is a W3C Recommendation.

XHTML stands for eXtensible HyperText Markup Language; it is a more strict, and more XML-like version of HTML, defined as an XML application, and supported by all major browsers

HTML stands for Hypertext Markup Language. In simple terms, it is a way of using symbols or codes in a file to tell web browsers how the words and images on the webpage should appear.

HTML5 stands for Hypertext Markup Language 5. It is a language used to create and display web pages. It helps developers build web applications that work on both desktop and mobile browsers.
